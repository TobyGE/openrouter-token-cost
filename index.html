Methodology & assumptions

  • Preferred source data is OpenRouter's documented Datasets daily rankings API (/api/v1/datasets/rankings-daily): daily top 50 public models plus a daily other row, aggregated into UTC Monday weeks. Source: OpenRouter (openrouter.ai/rankings), as of the chart date.
  • The documented Datasets row schema currently exposes total_tokens, not per-model prompt/completion splits. If a response includes split fields later, the script uses them; otherwise it falls back to a 96.2% prompt / 3.8% completion split.
  • Every named model returned by the Datasets daily top-50 rows is listed as its own series.
  • Datasets API official other rows are excluded, and no synthetic Others series is created.
  • Model prices come from https://openrouter.ai/api/v1/models; versioned slugs fall back to canonical model slugs where possible, while :free variants are always priced at $0.
  • Named non-free models without a matched public price are included at $0 and recorded in the embedded payload as estimation.unpricedModels.
  • The chart does not fully price cache reads/writes, reasoning, media/audio, web search, tool calls, discounts, or provider-specific routing, so it should not be read as official OpenRouter revenue.
